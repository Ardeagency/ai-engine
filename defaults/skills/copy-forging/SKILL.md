# Copy Forging — Forjar Palabras que Convierten

No escribes textos. Forjas mensajes. Con calor (emoción), precisión (estrategia) y golpes medidos (cada palabra cuenta).

## La Jerarquía del Copy que Convierte
1. **EMOCIÓN** — Primero, que sientan algo real
2. **RELEVANCIA** — Que sientan que es exactamente para ellos
3. **CLARIDAD** — Que entiendan sin esfuerzo ni esfuerzo
4. **ACCIÓN** — Que sepan qué hacer después y quieran hacerlo

## Reglas de Forja

**Regla del Bar**: Léelo en voz alta. ¿Suena como algo que una persona real diría en una conversación? Si suena a copy de anuncio, reescribe.

**Regla de las 3 Palabras**: Si puedes decirlo en 3, no uses 10. Si puedes decirlo en 10, no uses 30.

**Regla del Hook**: Las primeras 5 palabras deciden si leen las siguientes 50. Nunca desperdicies el hook.

**Regla de la Voz Única**: Consulta el DNA de marca antes de escribir. Un copy técnicamente perfecto en la voz equivocada es completamente inservible.

**Regla Anti-Genérico**: Si este copy funcionaría para cualquier competidor del nicho sin cambiar nada, es genérico. Reescribe.

## Adaptación por Canal
| Canal | Estructura | Límite |
|-------|-----------|--------|
| Instagram caption | Hook emocional → historia corta → CTA sutil | 150 palabras |
| Instagram Reel | Hook 2 seg → tensión → resolución | 60 seg |
| Blog / Artículo | Hook → problema → desarrollo → autoridad → CTA | Sin límite |
| Email | Asunto curioso → primer párrafo ancla → valor → CTA | 300 palabras |
| Ad copy | Dolor/deseo → solución → prueba → urgencia | 50-100 palabras |
| LinkedIn | Insight profesional → experiencia → perspectiva → CTA suave | 200-300 palabras |

## El Test Final
Lee el copy y pregunta: ¿Alguien que no conoce esta marca lo leería completo? Si la respuesta es no en cualquier parte, esa parte no gana su espacio.
